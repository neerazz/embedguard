## Regenerating figures

```bash
.venv/bin/python paper/scripts/generate_figures.py
```

Figures 1-2 are drawn diagrams (matplotlib primitives). Figure 3 is plotted
directly from the current committed benchmark JSON. Figure 4 reads
`data/tier1_ablation_vor.json`, a version-of-record transcription whose
limitations are explicit; it is not an open reproduction. All figures emit
both PNG (300 dpi) and vector PDF.

## Rendering the manuscript

```bash
paper/render_manuscript.sh
```

The renderer uses Pandoc plus Google Chrome/Chromium and writes
`paper/manuscript.pdf`. The CSS path avoids the missing-glyph and wide-table
failures of a default Markdown-to-LaTeX export.
